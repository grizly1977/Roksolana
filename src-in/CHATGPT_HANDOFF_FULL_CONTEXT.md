## 8. Data Model / Information Architecture

### Entity: Order
- **Fields:** season (August/January), year, total_USD, supplier (Turkish), line_items[]
- **Business rule:** August = winter (wool + diabetic + cotton); January = summer (diabetic + cotton only)
- **Key relationships:** Order → has many OrderLines

### Entity: OrderLine
- **Fields:** product_name, category, size_variant, quantity_dz (or quantity_pcs for tights), price_per_unit, total_USD, notes
- **Size variants for socks:** S/M, M/L (diabetic only), L/XL
- **Size variants for tights:** M, L, XL, 2XL, 3XL, 4XL
- **Units:** Socks = dozens (dz = 12 pairs); Tights = individual pieces (поштучно)

### Entity: Product
- **Fields:** name, category, wool_type, price_per_dz, size_options, status
- **Categories:** Wool Socks, Cotton Socks, Diabetic Socks, Tights
- **Product name convention:** "[Material] Wool Socks" or "[Material] Tights" — the material name is a brand/line name, not necessarily 100% that material
- **Special note:** "Zhytomyr" is a brand name for cotton socks (named after Ukrainian city)

### Entity: Supplier
- **Location:** Istanbul, Turkey
- **Relationship:** Single supplier for all products
- **Communication language:** Russian and Turkish (Vlad speaks both with supplier)
- **Visit schedule:** June 3, 2026 — one day only

### Entity: Sample / SampleRequest
- **Purpose:** Before each new product is mass-ordered, supplier prepares a sample
- **June 3, 2026 samples needed:**

  2. Mink (replicate from existing reference sample Vlad brought previously)

  5. Wool Tights M–4XL (full size run)
  6. Bamboo Tights M–4XL (full size run)

### Entity: Label / Packaging
- **Format:** SVG files
- **Dimensions:** 972px wide (standard label width from existing Lavender S-M.svg reference)
- **Front label height:** 1,740px
